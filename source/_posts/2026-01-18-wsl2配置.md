# 导出

使用wsl install和微软商店安装，默认是安装在C盘，可以将其导出为镜像，然后迁移到其他盘再导入，实现迁移

在其他文件夹下输入：wsl --export <发行版名称> <导出的文件名>   // 文件名后缀是tar

卸载：wsl --unregister Ubuntu

导入：wsl --import Ubuntu D:\WSL\Ubuntu(安装位置) D:\WSL\Ubuntu\ubuntu.tar 

设置默认启动发行版：wsl -s 你的发行版名

导入后默认用户为root，需要进行修改

```bash
vim /etc/wsl.conf
然后再里面添加以下信息即可

[user]
default=你的用户名
```

记得关闭wsl再启动，重启才会生效

## 使用

### 复制

尝试将checkhealth的结果复制出来，发现右键不行，显示缺少一个剪切板插件。。。需要在vim里面安装clipboard。

ai给了个简单的方案，就是win的clip.exe剪切板程序,wsl里面也能调用

```bash
将当前缓冲区所有内容输出到剪切板中，但是编码和win有点不兼容，emoji会变成乱码
:%w !clip.exe
```

将结果提供给clip.exe。

Vim内部有很多寄存器，每个都有名字，可以理解为剪切板，用来存各种复杂/删除的内容，我们可以指定从指定的寄存器来读取东西。

不加任何前缀，y、d、p会使用默认寄存器，这些内容不会影响系统的剪切板

常用的寄存器
- `”` 无名寄存器，默认寄存器
- `0` 只存最近一次复制的东西
- `a-z`"命名寄存器等。。
- `+` 系统寄存器
- `*` 选择缓冲区，在wsl里要通过win32yank，clip才能连接到win

### 三种模式

- 正常模式（Normal）：默认模式，用来移动、删除、复制等
- 插入模式（Insert）：像普通编辑器那样输入文字
- 可视模式（Visual）：选中一块文本

切换方法：

- 任何时候回到「正常模式」：`Esc`
- 正常模式 → 插入模式：`i`（在光标前插入）、`a`（在光标后）
- 正常模式 → 可视模式：`v`

### 退出


- 正常模式输入（先按`Esc`保在 Normal）：

  - `:q` 回车：退出当前文件
  - `:q!` 回车：不保存强制退出
  - `:w` 回车：保存
  - `:wq` / `:x` 回车：保存并退出

### 编辑操作

在正常模式下：

- 插入: 
  - `o/O`：在光标下方/上方插入一行空行并进入插入模式

- 删除
  - `x`：删掉光标所在字符
  - `dw`：删到下一个单词开头
  - `dd`：剪切整行
  - `3dd`:剪切3行
- 撤销 / 重做
  - `u`：撤销
  - `Ctrl + r`：重做
- 选择模式（v字符选择模式，V行选择模式，ctrl v/q 进入矩形选择模式))
- 复制 + 粘贴
  - `y`：复制选中的东西，配合选择模式
  - `yy`：复制一整行
  - `3yy`: 复制当前行开始的3行
  - `yw`： 从光标开始复制这个单词
  - `y$ y0` 光标到行尾  光标到开头
  - `p`：在光标后粘贴
  - `P`：在光标前粘贴
- 搜索
  - `/关键词` 回车：向下搜索
  - `n` / `N`：下一个 / 上一个匹配



### 方向键

不使用小键盘的上下左右，使用

在正常模式下：

- `h`：左
- `j`：下
- `k`：上
- `l`：右

前面可以添加数字前缀来实现快速移动

常用跳转：

- `0`：到行首
- `$`：到行尾
- `gg`：到文件开头
- `G`：到文件结尾
- `w`：下一个单词开头
- `b`：上一个单词开头

### 命令模式

- 保存： `:w`
- 退出： `:q`
- 保存并退出： `:wq`
- 强制不保存退出： `:q!`
